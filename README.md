# Система — лендинг

Презентационный сайт платформы «Система Молодцова».
Статический сайт без сборки: HTML + CSS + JS, готов к деплою на GitHub Pages или любой статический хостинг.

## Структура

```
landing/
├── index.html               # Главная: hero, направления, возможности, программы, автор
├── directions/<slug>/       # Страницы 6 направлений (мини-презентации)
│   ├── selfstudy/           # Понять себя глубже
│   ├── calm/                # Спокойствие
│   ├── body/                # Тело и симптомы
│   ├── relationships/       # Отношения
│   ├── selfworth/           # Самооценка и опора
│   └── communication/       # Коммуникация и конфликты
├── features/<slug>/         # Страницы 5 возможностей (мини-презентации)
│   ├── ai/                  # Лиза — AI-помощник
│   ├── chat/                # Общий чат
│   ├── meditations/         # Медитации
│   ├── shorts/              # Shorts
│   └── marathons/           # Марафоны
├── css/style.css            # Дизайн-система (тёмная, в ДНК приложения)
├── js/main.js               # Reveal-анимации, навбар, мобильное меню, marquee
├── assets/                  # Картинки и лого (из основного проекта)
├── generate-pages.mjs       # Генератор страниц направлений и возможностей
└── .nojekyll                # Отключает Jekyll на GitHub Pages
```

## Правки внутренних страниц

Контент всех страниц лежит в одном месте — массивы `DIRECTIONS` и `FEATURES`
в `generate-pages.mjs`. После правок перегенерировать:

```bash
node generate-pages.mjs
```

## Локальный просмотр

Из папки `landing`:

```bash
npx serve .
# или
python -m http.server 8080
```

## Деплой на GitHub Pages

1. Создать отдельный репозиторий, запушить содержимое `landing/` в корень.
2. Settings → Pages → Deploy from a branch → `main` / `(root)`.
3. Все пути относительные — сайт работает и на `user.github.io/repo/`,
   и на собственном домене (Settings → Pages → Custom domain).

## Важные ссылки

- Кнопки «Войти в систему» ведут на основной продукт: https://система-молодцова.рф/
- Адрес продукта задаётся в `generate-pages.mjs` (константа `MAIN_SITE`)
